# SNO Quadro — Поиск фото товаров из Excel

Программа читает Excel-файл прихода (`.xls` / `.xlsx`), извлекает товары из столбца **«Товар»**, парсит название (бренд, модель, цвет) и автоматически ищет и скачивает фото каждого товара в хорошем качестве на белом фоне.

## Установка

```bash
pip install -r requirements.txt
```

## Использование

### Базовый запуск
```bash
python search_images.py приход6.xls
```

Фото будут сохранены в папку `images/` с именами `Бренд_Модель_Цвет.jpg`.

### Параметры

| Параметр | Описание | По умолчанию |
|----------|----------|--------------|
| `--output`, `-o` | Папка для сохранения фото | `images` |
| `--count`, `-c` | Количество фото на товар | `1` |
| `--delay`, `-d` | Задержка между запросами (сек) | `1.0` |
| `--dry-run` | Показать товары без скачивания | — |

### Примеры
```bash
# Скачать по 3 фото каждого товара в папку "фото"
python search_images.py приход6.xls -o фото -c 3

# Только посмотреть какие товары нашлись в файле
python search_images.py приход6.xls --dry-run

# С увеличенной задержкой (если блокируют)
python search_images.py приход6.xls --delay 3
```

## Формат входного файла

Программа ожидает Excel-файл со столбцом **«Товар»** (или «Наименование»):

| № | Товар | Количество | Цена |
|---|-------|------------|------|
| 1 | Chopard 358V 700 м/о | 2 | 15000 |
| 2 | Chopard 402S 700 м/о | 1 | 18000 |
| 3 | Ray-Ban RB3025 001 | 5 | 12000 |

Программа автоматически:
- Ищет строку-заголовок (не обязательно первая строка)
- Парсит «Бренд Модель Цвет» из названия
- Убирает суффиксы (м/о, ж/о, унисекс)
- Дедуплицирует одинаковые товары
